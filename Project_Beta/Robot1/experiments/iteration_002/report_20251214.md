## Issues Identified

### 1. Distribution Shift Problem
The model was trained on "moving" images from rule-based driving. At race start (stationary), the model sees unfamiliar images and outputs near-zero torque.

**Mitigation Applied**: Start Boost (40 frames, MIN_TORQUE=0.4)

### 2. Over-Steering
The model outputs extreme steering values (~0.5 rad) in curves, causing over-rotation.

**Mitigation Applied**:
- MAX_STEER = 0.25 rad (~14°)
- Steering smoothing filter (alpha=0.6)

### 3. Excessive Speed
High torque values (0.7-0.9) cause wall collisions in curves.

**Mitigation Applied**: MAX_DRIVE_TORQUE = 0.55

### 4. Training Data Quality
Rule-based driving data may contain aggressive steering patterns that the model learns and amplifies.
